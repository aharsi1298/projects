Project on console


To do this project you need Visual Studio Code. Click here and download it. Then install and open it up. Create a folder called "ADV-C77" and open it with VSCode



The console screen where only text can be stored. It is used for debugging. On CodePen, you can click on the 'Console' button and in Chrome, click Ctrl+Shift+I to open the console. To print, which means display text, onto the console, you need to use the console.log() function.
